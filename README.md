# APCSA Freedom Project
by **Robert Xu**

## Context
Hello, I'm a student at [HSTAT](https://www.hstat.org/) in the [Software Engineering Program](https://hstatsep.github.io/). All the students in APCSA did a "Freedom Project" which is a year-long project all about making anything using technology.

For my project, I chose to tinker and use the tool [**Godot**](https://godotengine.org/) in order to help me make a game that's about farming.
